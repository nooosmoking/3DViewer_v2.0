Description of the work of the 3DViewer 2.0 program

Здесь вы найдёте краткое описание работы программы 3DViewer 2.0 и её функционала.

Программа предоставляет возможность:

Загружать каркасную модель из файла формата obj (поддержка списка вершин и поверхностей).

Перемещать модель на заданное расстояние относительно осей X, Y, Z.

Поворачивать модель на заданный угол относительно своих осей X, Y, Z.

Масштабировать модель на заданное значение.

Графический пользовательский интерфейс содержит:

Кнопку для выбора файла с моделью.

Зону визуализации каркасной модели.

Органы управления перемещением модели.

Органы управления поворотом модели.

Органы управления масштабированием модели.

Информацию о загруженной модели - название файла, кол-во вершин и фейсетов.
